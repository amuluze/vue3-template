# `Vue3` `Template`

`vue3-template` 是一个基于 `element-plus` 的免费开源的中后台模板，使用了最新的 `Vue3`、`Vite`、`TypeScript` 等主流技术。开箱即用的中后台前端解决方案，
可以用来作为项目的启动模板，也可以用于学习参考。

## 主要功能

- [x] 布局 `layout`
- [x] 路由 `router`
- [x] 状态管理 `store`
- [x] 请求 `axios`
- [x] 主题 `theme`
- [X] 国际化 `i18n`
- [X] 组件 `components`
    - [x] SVG 图标 `svg-icon`
    - [x] 可视化 `echarts`
    - [x] 表格 `el-table`
    - [x] 表单 `el-form`
    - [x] 弹窗 `el-dialog`
    - [x] 通知 `el-notification`
    - [x] 消息弹窗 `Message`

## 前序准备
- `node` 和 `git` - 项目开发环境
- `Vite` - 熟悉 `vite` 特性
- `Vue3` - 熟悉 `Vue` 基础语法
- `TypeScript` - 熟悉 `TypeScript` 基本语法
- `Es6+` - 熟悉 `es6` 基本语法
- `Vue-Router` - 熟悉 `vue-router` 基本使用
- `Element-Plus` - `element-plus` 基本使用


## 安装及使用

- 获取代码
```bash
git clone https://github.com/kailong321200875/vue-element-plus-admin.git
```

- 安装依赖
```bash
make install
```

- 运行
```bash
make dev
```
- 打包
```bash
make build
```
